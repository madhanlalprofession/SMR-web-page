Technologies used in Java

Technology Description
Java SE(Standard Edition) Core Java libraries for general-purpose programming.
Java EE(Enterprise Edition) For enterprise-level applications, includes APIs for web services, security, and messaging.
Java ME(Micro Edition) Used for developing applications on small devices like mobile phones.
JVM(Java Virtual Machine) The engine that runs Java bytecode.
JDK (Java Development Kit) The full Java development kit including JRE and other tools for Java development.
JRE (Java Runtime Environment) Provides libraries and Java Virtual Machine to run Java applications.
